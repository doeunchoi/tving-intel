      STEP 4 신작 콘텐츠 추가 Claude AI 자동완성
      두 가지 방법 중 선택
      🤖 방법 A. AI 자동완성 (권장)
      프로그램명만 입력하면 장르·시놉시스·타겟·광고상품을 AI가 자동으로 채워줌. 확인 후 수정하고 저장.
      ✏️ 방법 B. 직접 입력
      "직접 추가" 버튼 클릭 → 빈 폼이 열림 → 각 항목 직접 입력. Google Sheets 연동 중이라면 시트에 행 추가 후 "데이터 동기화"가 더 편함.
      AI 자동완성은 인터넷 검색 기반으로 정보를 채웁니다. 시놉시스·타겟·출연진은 반드시 확인 후 수정하세요.
      STEP 5 썸네일 이미지 관리 TMDB 자동 + 수동 보완
      썸네일 자동화 원리
      1. TMDB 자동 매칭 — CONFIG에 TMDB API Key가 설정된 경우, 프로그램명으로 자동 검색하여 포스터 이미지를 표시. 한국 OTT 콘텐츠의 약 70~80%가 자동 매칭됨.
      2. 자동 매칭 실패 시 — 색상 플레이스홀더(타이틀 이니셜)로 표시됨. 이 경우 아래에서 수동으로 URL을 입력하면 됨.
      3. TMDB API Key 발급 — themoviedb.org/settings/api 에서 무료 발급 → 파일 상단 CONFIG.TMDB_KEY에 입력
      썸네일 URL 일괄 입력
      형식 1: 타이틀 | 이미지URL
      형식 2: 타이틀[탭키]이미지URL (구글 시트에서 복사 붙여넣기)
      · 공개 접근 가능한 이미지 URL만 동작 (로그인 필요 URL 불가)
      · 세로형 2:3 비율 권장 (예: 300×450px, 500×750px)
      이미지 URL 찾는 방법
      방법 1. TMDB에서 직접 복사
      themoviedb.org → 프로그램 검색 → 포스터 이미지 → 우클릭 → "이미지 주소 복사"

      방법 2. 구글 이미지 검색
      "[프로그램명] 포스터" 검색 → 이미지 클릭 → 원본 이미지 URL 복사

      방법 3. 공식 채널 활용
      TVING 또는 Wavve 공식 보도자료·SNS의 이미지를 imgur.com 등에 업로드 후 URL 사용
      STEP 6 광고 집행 벤치마크 & 사례 데이터 인증 후 열람
      모달 "광고 집행" 탭에 표시되는 데이터 종류
      📊 KPI 벤치마크 (장르별 평균)
      · VCR (Video Completion Rate) — 광고 완주율
      · CTR (Click-Through Rate) — 클릭률
      · 브랜드 리프트 — 인지도/호감도 상승폭
      → 출처: 태블로 캠페인 성과 대시보드 (장르별 집계값)
      📋 집행 사례 (브랜드별 케이스)
      · 브랜드명 (익명처리 가능)
      · 주요 성과 요약 1줄
      · 집행 방식 요약 1줄
      → 출처: 광고영업팀 / 인사이트팀 케이스 스터디
      🔄 자동 연동 방향 (장기)
      권장 태블로 Embedded Analytics — 태블로 서버에서 "장르별 KPI 뷰"를 공개 embed URL로 발행 → 코드에서 장르값으로 iframe URL을 동적 생성. 라이선스가 임베드를 지원하는지 데이터팀 확인 필요.
      대안 Google Sheets API 자동화 — 태블로에서 장르별 KPI를 CSV/Excel로 정기 추출 → Sheets에 붙여넣기 → 이 플랫폼이 Sheets에서 읽어옴. 수동이지만 월 1회 작업으로 데이터 최신화 가능.
      현재 수동 직접 입력 — 아래 가이드 참조. 장르별 평균값을 파일 코드에 직접 기입.
      📝 당분간 수동 관리 방법 — Google Sheets 벤치마크 탭 구조
      라인업 Sheets에 "벤치마크" 시트 탭을 추가하고 아래 컬럼 구조로 입력하세요.
      이 플랫폼은 콘텐츠의 genre 값으로 해당 행을 찾아 KPI를 표시합니다.
      열 항목 입력 형식 예시
      Agenre장르명 (라인업 C열과 동일하게)드라마
      BvcrVCR 상승폭 (숫자만, % 기호 제외)42
      Cctr평균 CTR (숫자만, 소수점 가능)1.8
      Dlift브랜드 리프트 배수 (숫자만)3.5
      Ecase1_brand사례 1 브랜드명 (익명 처리 가능)뷰티 브랜드 A
      Fcase1_result성과 요약 1줄브랜드 인지도 23%↑, CTR 1.9%
      Gcase1_detail집행 방식 요약 1줄Pre-roll 4주 + 중간광고, 25-34세 여성 타겟팅
      H-Jcase2_*사례 2 (brand / result / detail)동일 구조 반복
      📌 태블로에서 데이터 추출하는 방법
      1. 태블로 캠페인 성과 대시보드 접속
      2. 장르별 필터 적용 → 기간: 최근 12개월 or 전체 기간
      3. VCR · CTR · 브랜드 리프트 집계값 메모 (장르당 평균값)
      4. 위 Sheets 벤치마크 탭에 장르별로 1행씩 입력
      5. 라인업 Sheets와 같은 스프레드시트에 탭으로 추가하면 API 연동 시 한 번에 읽어올 수 있음
      → 권장 업데이트 주기: 분기 1회 (캠페인 성과가 충분히 쌓인 시점)
      현재 대시보드에 표시되는 KPI(VCR ↑38%, CTR 1.6%, 브랜드 리프트 3.2×)는 샘플 기본값입니다. Sheets 벤치마크 탭 연동 전까지는 파일 내 CASES 객체와 d.benchmark를 직접 수정하거나, 실제 데이터를 Sheets에 입력 후 "데이터 동기화"를 누르면 자동 반영됩니다.
